# PDFmerger
PDF merger built with Python, Tkinter and PyPDF2.<br>


### Screenshots

<img src="/img/screen1.jpg" width="400">

### Setup
| Name    | Version |
| :--:    | :--:    |
| Python  | 3.6.5   |
| Tkinter | 8.6     |
| PyPDF2  | 1.26.0  |

Download and run with `python merge.py`. Or download executable file `merge.exe`
